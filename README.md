### Free-rider-Problem-a-New-Perspective-in-Game-Theory
- Engineered a mathematical framework to analyze free-riding behavior in collaborative settings using Game Theory; utilized payoff matrices and dominant strategy equilibrium to model decision-making processes between actors.
- Conducted large-scale simulations on 10,000+ data pairs to quantify the "Free-Riding Index"; identified that aligning individual efficiency and time-cost perceptions can significantly enhance collective productivity in academic and corporate environments.
